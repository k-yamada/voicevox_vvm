# VOICEVOX VVM とは

VOICEVOX 用の 音声モデル（VVM ファイル）です。
[VOICEVOX CORE](https://github.com/VOICEVOX/voicevox_core)を用いることで、プログラム上で VOICEVOX 音声を生成できます。
